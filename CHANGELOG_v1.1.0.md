## 🎯 Новые возможности

### 1. Custom API Token
Используйте свой токен вместо стандартного Anthropic API ключа.

**Применение:**
- Токены от OmniRoute
- Прокси с собственной аутентификацией
- Корпоративные gateway
- Альтернативные провайдеры

**Пример:**
```
sk-ant-api03-xxxxxxxxxxxxx
```

---

### 2. Custom Base URL
Измените endpoint API на свой.

**Применение:**
- Локальный gateway (OmniRoute на localhost)
- Прокси-серверы
- Корпоративные API endpoints
- Альтернативные провайдеры с Anthropic-совместимым API

**Примеры:**
```
http://localhost:20128/v1
https://api.omniroute.ai
https://your-proxy.com/api/v1
```

---

### 3. Custom Model Name
Укажите специфичное имя модели для вашего gateway.

**Применение:**
- Gateway требует специфичное имя
- Кастомные модели
- Альтернативные названия моделей

**Примеры:**
```
kr/claude-sonnet-4.5
custom-model-name
gpt-4-compatible
```

---

### 4. Disable Experimental Betas
Отключите экспериментальные функции Claude.

**Применение:**
- Gateway не поддерживает beta-функции
- Нужна максимальная совместимость
- Ошибки с beta-заголовками

---

### 5. Визуальный индикатор
Желтый значок ⚙️ в правом верхнем углу показывает, что используются кастомные настройки.

**Функции:**
- Автоматическое появление при активных настройках
- Клик открывает страницу настроек
- Работает во всех интерфейсах

---

### 6. Генерация .bat файла
Автоматическое создание эквивалентного батника в реальном времени.

**Возможности:**
- Живой предпросмотр
- Копирование кода
- Комментарии для отключенных опций

## 🚀 Как использовать

### Быстрый старт

1. Откройте `index.html`
2. Нажмите на желтую карточку "Кастомные настройки"
3. Включите нужные опции (чекбоксы)
4. Заполните поля
5. Нажмите "Сохранить настройки"
6. Откройте любой интерфейс - настройки применятся автоматически!

---

## 💡 Примеры использования

### Пример 1: OmniRoute локально

```
✓ Использовать кастомный токен
  Token: your-omniroute-token-here

✓ Использовать кастомный endpoint
  URL: http://localhost:20128/v1

✓ Использовать кастомное имя модели
  Model: kr/claude-sonnet-4.5

✓ Отключить experimental betas
```

**Эквивалент .bat:**
```batch
@echo off
setlocal

set "ANTHROPIC_AUTH_TOKEN=your-omniroute-token-here"
set "ANTHROPIC_BASE_URL=http://localhost:20128/v1"
set "ANTHROPIC_MODEL=kr/claude-sonnet-4.5"
set "CLAUDE_CODE_DISABLE_EXPERIMENTAL_BETAS=1"

echo Starting Claude Code...
claude

endlocal
```

## 🔒 Безопасность

⚠️ **Важные замечания:**

1. **Хранение данных:**
   - Настройки сохраняются в localStorage браузера
   - Токены хранятся в открытом виде
   - Данные доступны только в этом браузере

2. **Рекомендации:**
   - Не используйте на общедоступных компьютерах
   - Очищайте localStorage после использования на чужих устройствах
   - Регулярно меняйте токены

3. **Очистка данных:**
   ```javascript
   // Полная очистка всех настроек
   localStorage.removeItem('claude_custom_settings');
   ```

## 🐛 Известные ограничения

1. **localStorage:**
   - Лимит ~5-10MB в зависимости от браузера
   - Очищается в режиме инкогнито

2. **Совместимость:**
   - Требуется современный браузер с поддержкой ES6+
   - Не работает в IE11

3. **Безопасность:**
   - Токены хранятся в открытом виде
   - Нет шифрования данных
