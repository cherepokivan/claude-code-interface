Настройка кастомных параметров для Claude API

ℹ️ Для чего это нужно?

Эти настройки позволяют использовать кастомные endpoints (например, OmniRoute, прокси-серверы) вместо стандартного Anthropic API. Полезно для локальных gateway или альтернативных провайдеров.

🔑 API Token
Например: sk-ant-api03-xxxxxxxxxxxxx
🌐 Base URL
Например: http://localhost:20128/v1 или https://api.omniroute.ai
🤖 Модель
Например: kr/claude-sonnet-4.5 или custom-model-name
🔌 Режим подключения
Подключиться к уже запущенному claude.exe в консоли (обычно http://localhost:3000)
🧪 Экспериментальные функции
Включите, если ваш gateway не поддерживает экспериментальные функции

⚠️ Внимание: Эти настройки сохраняются в localStorage браузера. Не используйте на общедоступных компьютерах!

📋 Эквивалент в .bat файле
@echo off setlocal REM Стандартные настройки Anthropic set "ANTHROPIC_AUTH_TOKEN=your-api-key" set "ANTHROPIC_BASE_URL=https://api.anthropic.com" echo Starting Claude Code... claude endlocal
