🔑 Получение API ключа

Для работы с интерфейсом вам нужен API ключ от Anthropic:

  • Зарегистрируйтесь на console.anthropic.com
  • Перейдите в раздел API Keys
  • Создайте новый ключ
  • Скопируйте и сохраните его

⚙️ Кастомные настройки

Используете OmniRoute, локальный gateway или прокси? Настройте кастомные параметры:

  • Custom API endpoint (например, http://localhost:20128/v1)
  • Custom токен для вашего gateway
  • Custom имя модели (например, kr/claude-sonnet-4.5)
  • Отключение experimental betas
