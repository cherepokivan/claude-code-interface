# 🚀 Быстрый старт - Claude Code Interface

## За 3 минуты до первого сообщения!

### Шаг 1: Откройте интерфейс (30 секунд)

**Windows:**
```
Двойной клик на start.bat
```

**Linux/macOS:**
```bash
chmod +x start.sh
./start.sh
```

**Или просто:**
```
Откройте index.html в браузере
```

---

### Шаг 2: Получите API ключ (2 минуты)

1. Перейдите на https://console.anthropic.com/
2. Войдите или зарегистрируйтесь
3. Откройте раздел **API Keys**
4. Нажмите **Create Key**
5. Скопируйте ключ (показывается только один раз!)

---

### Шаг 3: Настройте и начните (30 секунд)

1. В интерфейсе нажмите **⚙️** (настройки)
2. Вставьте API ключ
3. Нажмите **Сохранить**
4. Готово! Начните общение 🎉

## ⚠️ Частые проблемы

### "API Error" или "Invalid API Key"
✅ Проверьте правильность ключа  
✅ Убедитесь что ключ активен  
✅ Проверьте баланс аккаунта

### Интерфейс не загружается
✅ Проверьте интернет  
✅ Очистите кэш (Ctrl+Shift+Del)  
✅ Попробуйте другой браузер

### Сообщения не отправляются
✅ Откройте консоль (F12)  
✅ Проверьте настройки API  
✅ Убедитесь что JavaScript включен
