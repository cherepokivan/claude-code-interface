## 🚀 Как начать использовать

### Вариант 1: Через главную страницу
```
1. Откройте index.html в браузере
2. Выберите нужный интерфейс
3. Настройте API ключ
4. Начните общение!
```

### Вариант 2: Через лаунчер
```
Windows:
  start.bat

Linux/macOS:
  chmod +x start.sh
  ./start.sh
```

### Вариант 3: Напрямую
```
Откройте любой HTML файл в браузере:
- claude-interface.html (базовый)
- claude-advanced.html (продвинутый)
- claude-dark.html (темная тема)
- claude-mobile.html (мобильный)
```

---

## 🔑 Получение API ключа

1. Перейдите на https://console.anthropic.com/
2. Зарегистрируйтесь или войдите
3. Откройте раздел **API Keys**
4. Создайте новый ключ
5. Скопируйте и сохраните его
