## 🚀 Быстрый старт

### Шаг 1: Скачайте файлы
Все HTML файлы готовы к использованию без установки.

### Шаг 2: Откройте в браузере
Просто дважды кликните на любой `.html` файл или перетащите его в браузер.

1. Нажмите на иконку настроек ⚙️
2. Вставьте ваш API ключ
3. Нажмите "Сохранить"

Введите сообщение и нажмите Enter или кнопку отправки.

## 🔧 Детальная установка

### Требования
- Современный браузер (Chrome, Firefox, Safari, Edge)
- Интернет-соединение
- API ключ от Anthropic

### Поддерживаемые браузеры
- ✅ Google Chrome 90+
- ✅ Mozilla Firefox 88+
- ✅ Safari 14+
- ✅ Microsoft Edge 90+
- ✅ Opera 76+

### Установка на разные платформы

#### Windows
1. Скачайте все файлы в одну папку
2. Дважды кликните на нужный `.html` файл
3. Файл откроется в браузере по умолчанию

#### macOS
1. Скачайте файлы
2. Откройте через Safari или Chrome
3. Для быстрого доступа добавьте в Dock

#### Linux
1. Скачайте файлы
2. Откройте через Firefox или Chrome
3. Можно создать desktop shortcut

#### iOS (iPhone/iPad)
1. Откройте `claude-mobile.html` в Safari
2. Нажмите кнопку "Поделиться"
3. Выберите "На экран Домой"
4. Теперь работает как приложение!

#### Android
1. Откройте `claude-mobile.html` в Chrome
2. Меню → "Добавить на главный экран"
3. Работает как нативное приложение

## 🔑 Настройка API

### Получение API ключа

1. **Регистрация**
   - Перейдите на https://console.anthropic.com/
   - Создайте аккаунт или войдите

2. **Создание ключа**
   - Откройте раздел "API Keys"
   - Нажмите "Create Key"
   - Скопируйте ключ (он показывается только один раз!)

3. **Настройка в интерфейсе**
   - Откройте любой интерфейс
   - Нажмите ⚙️ (настройки)
   - Вставьте ключ в поле "API Key"
   - Нажмите "Сохранить"

### Выбор модели

**Claude Sonnet 4.6** (рекомендуется)
- Баланс скорости и качества
- Подходит для большинства задач
- Оптимальная цена

**Claude Opus 4.7**
- Максимальная производительность
- Сложные задачи
- Более высокая стоимость

**Claude Haiku 4.5**
- Самая быстрая модель
- Простые задачи
- Минимальная стоимость

### Настройка параметров

**Max Tokens** (Максимум токенов)
- Минимум: 256
- Рекомендуется: 4096
- Максимум: 8192
- Больше токенов = более длинные ответы

**Temperature** (только в advanced версии)
- 0.0 = детерминированные ответы
- 1.0 = креативные ответы
- Рекомендуется: 1.0

## 💡 Использование

### Базовые команды

**Отправка сообщения**
- Введите текст
- Нажмите Enter (или Shift+Enter для новой строки)
- Или нажмите кнопку ➤

**Форматирование кода**
```python
# Код автоматически форматируется
def hello():
    print("Hello, Claude!")
```

**Копирование ответа**
- Наведите на сообщение Claude
- Нажмите "📋 Копировать"

### Продвинутые функции

**Управление разговорами** (claude-advanced.html)
- Создание нового чата: кнопка "+ Новый чат"
- Переключение между чатами: клик по разговору в боковой панели
- Автоматическое сохранение

**Экспорт чатов**
- Нажмите 💾 (экспорт)
- Чат сохранится как `.txt` файл
- Можно открыть в любом текстовом редакторе

**Статистика** (claude-advanced.html)
- Нажмите 📊
- Просмотрите количество сообщений и токенов

**Очистка истории**
- Нажмите 🗑️
- Подтвердите действие
- История удалится из localStorage

## 🔧 Решение проблем

### Проблема: "API Error" или "Invalid API Key"

**Решение:**
1. Проверьте правильность API ключа
2. Убедитесь, что ключ активен в консоли Anthropic
3. Проверьте баланс аккаунта
4. Попробуйте создать новый ключ

### Проблема: Интерфейс не загружается

**Решение:**
1. Проверьте интернет-соединение
2. Очистите кэш браузера (Ctrl+Shift+Del)
3. Попробуйте другой браузер
4. Отключите блокировщики рекламы

### Проблема: Сообщения не отправляются

**Решение:**
1. Проверьте настройки API
2. Откройте консоль браузера (F12)
3. Проверьте ошибки в консоли
4. Убедитесь, что JavaScript включен

### Проблема: История не сохраняется

**Решение:**
1. Проверьте, что localStorage включен
2. Не используйте режим инкогнито
3. Проверьте настройки приватности браузера
4. Очистите localStorage и попробуйте снова

### Проблема: Медленная работа

**Решение:**
1. Используйте модель Haiku для быстрых ответов
2. Уменьшите Max Tokens
3. Очистите старые разговоры
4. Проверьте скорость интернета

### Проблема: Мобильная версия не работает

**Решение:**
1. Используйте `claude-mobile.html`
2. Обновите браузер
3. Проверьте поддержку JavaScript
4. Попробуйте другой браузер

## ❓ FAQ

### Нужен ли сервер для работы?
Нет, все работает полностью в браузере.

### Безопасно ли хранить API ключ в браузере?
API ключ хранится в localStorage. Не используйте на общедоступных компьютерах.

### Можно ли использовать без интернета?
Нет, требуется подключение к API Anthropic.

### Сколько стоит использование?
Зависит от выбранной модели и количества токенов. Проверьте цены на https://www.anthropic.com/pricing

### Можно ли использовать несколько API ключей?
Да, но нужно переключать их в настройках вручную.

### Работает ли на старых браузерах?
Требуется современный браузер с поддержкой ES6+.

### Можно ли кастомизировать интерфейс?
Да, все файлы HTML/CSS/JS открыты для редактирования.

### Есть ли лимиты на количество сообщений?
Лимиты определяются вашим планом Anthropic API.

### Можно ли использовать для коммерческих проектов?
Да, но проверьте условия использования Anthropic API.

### Как обновить интерфейс?
Просто замените старые HTML файлы на новые.
