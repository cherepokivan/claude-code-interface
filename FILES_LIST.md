- **Назначение:** Главная страница выбора интерфейса
- **Функции:**

### 2. **claude-interface.html** (~16 KB)
- **Назначение:** Базовый интерфейс для начинающих
- **Функции:**
  - Простой чат с Claude
  - Настройки API ключа
  - Форматирование кода
  - Сохранение истории
  - Минималистичный дизайн
- **Когда использовать:** Первое знакомство, простые задачи

### 3. **claude-advanced.html** (~27 KB)
- **Назначение:** Продвинутый интерфейс с полным функционалом
- **Функции:**
  - Управление несколькими разговорами
  - Боковая панель с историей
  - Статистика использования (сообщения, токены)
  - Экспорт чатов в TXT
  - Копирование сообщений
  - Настройка параметров модели (temperature, max tokens)
  - Выбор модели Claude
- **Когда использовать:** Работа с несколькими проектами, нужна статистика

### 4. **claude-dark.html** (~27 KB)
- **Назначение:** Интерфейс с темной темой
- **Функции:**
  - Темная цветовая схема (#0f172a фон)
  - Подсветка синтаксиса кода
  - Быстрые предложения (4 карточки)
  - Экспорт чатов
  - Очистка истории
  - Удобно для глаз в темноте
- **Когда использовать:** Работа ночью, программирование, длительные сессии

### 5. **claude-mobile.html** (~20 KB)
- **Назначение:** Мобильная версия для смартфонов
- **Функции:**
  - Touch-friendly интерфейс
  - Боковое выдвижное меню
  - PWA поддержка (можно добавить на главный экран)
  - Safe area insets для iPhone
  - Оптимизация для iOS/Android
  - Компактный дизайн
- **Когда использовать:** На телефоне, планшете, в дороге
