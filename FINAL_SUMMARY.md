## 🚀 Быстрый старт

### Вариант 1: Стандартное использование

1. Откройте **index.html** в браузере
2. Выберите интерфейс (базовый, продвинутый, темный или мобильный)
3. Получите API ключ на https://console.anthropic.com/
4. Настройте ключ в интерфейсе (⚙️)
5. Начните работу с Claude!

### Вариант 2: С кастомными настройками (OmniRoute, прокси)

1. Откройте **index.html** в браузере
2. Нажмите на желтую карточку **"Кастомные настройки"**
3. Включите нужные опции (чекбоксы)
4. Заполните поля:
   - Token: `sk-33cc6a3db50d1ef7-2a4327-c0bc7138`
   - URL: `http://localhost:20128/v1`
   - Model: `kr/claude-sonnet-4.5`
5. Нажмите **"Сохранить настройки"**
6. Откройте любой интерфейс - настройки применятся автоматически!

## 💡 Примеры использования

### Пример 1: OmniRoute локально

```
Настройки:
✓ Custom Token:  your-omniroute-token-here
✓ Custom URL:    http://localhost:20128/v1
✓ Custom Model:  kr/claude-sonnet-4.5
✓ Disable Betas: Включено
```

**Эквивалентный .bat файл:**
```batch
@echo off
setlocal

set "ANTHROPIC_AUTH_TOKEN=your-omniroute-token-here"
set "ANTHROPIC_BASE_URL=http://localhost:20128/v1"
set "ANTHROPIC_MODEL=kr/claude-sonnet-4.5"
set "CLAUDE_CODE_DISABLE_EXPERIMENTAL_BETAS=1"

echo Starting Claude Code...
claude

endlocal
```

### Пример 2: Корпоративный прокси

```
Настройки:
✓ Custom Token:  company-api-key-12345
✓ Custom URL:    https://api.company.com/claude/v1
□ Custom Model:  (используется стандартное)
□ Disable Betas: (не требуется)
```

### Пример 3: Стандартный Anthropic API

```
Настройки:
□ Все опции отключены
Используется стандартный API ключ из настроек интерфейса
```

## 🔒 Безопасность

⚠️ **Важно:**
- API ключи и настройки хранятся в localStorage браузера
- Токены хранятся в открытом виде
- Не используйте на общедоступных компьютерах
- Регулярно обновляйте ключи
- Очищайте localStorage после использования на чужих устройствах

**Очистка данных:**
```javascript
// Очистить все настройки
localStorage.clear();

// Или только кастомные настройки
localStorage.removeItem('claude_custom_settings');
```
