## 📊 Детальное описание файлов

### 🌐 HTML Интерфейсы

#### **index.html** (Главная страница)
- **Размер:** ~8 KB
- **Назначение:** Стартовая страница для выбора интерфейса
- **Функции:**
  - Карточки всех доступных интерфейсов
  - Быстрый запуск любой версии
  - Ссылки на документацию
  - Информация о проекте

#### **claude-interface.html** (Базовый)
- **Размер:** ~15 KB
- **Назначение:** Простой интерфейс для начинающих
- **Функции:**
  - Базовый чат с Claude
  - Настройки API ключа
  - Форматирование кода
  - Сохранение истории
- **Технологии:** HTML5, CSS3, Vanilla JS
- **Зависимости:** Нет

#### **claude-advanced.html** (Продвинутый)
- **Размер:** ~25 KB
- **Назначение:** Полнофункциональный интерфейс
- **Функции:**
  - Управление несколькими разговорами
  - Боковая панель с историей
  - Статистика использования
  - Экспорт чатов
  - Копирование сообщений
  - Настройка параметров модели
- **Технологии:** HTML5, CSS3 Grid/Flexbox, ES6+
- **Зависимости:** Нет

#### **claude-dark.html** (Темная тема)
- **Размер:** ~20 KB
- **Назначение:** Интерфейс для работы в темное время
- **Функции:**
  - Темная цветовая схема
  - Подсветка синтаксиса кода
  - Быстрые предложения
  - Экспорт чатов
  - Очистка истории
- **Цвета:** #0f172a (фон), #8b5cf6 (акцент)
- **Технологии:** HTML5, CSS3, ES6+

#### **claude-mobile.html** (Мобильная версия)
- **Размер:** ~18 KB
- **Назначение:** Оптимизация для смартфонов
- **Функции:**
  - Touch-friendly интерфейс
  - Боковое выдвижное меню
  - PWA поддержка
  - Адаптивная верстка
  - Оптимизация для iOS/Android
- **Особенности:**
  - Safe area insets для iPhone
  - Viewport meta для мобильных
  - Отключение масштабирования
- **Технологии:** HTML5, CSS3, Mobile-first JS

## 🚀 Инструкция по развертыванию

### Локальное использование

#### Вариант 1: Прямое открытие
```bash
# Просто откройте любой HTML файл в браузере
# Двойной клик на файле или
start index.html  # Windows
open index.html   # macOS
xdg-open index.html  # Linux
```

#### Вариант 2: Через лаунчер
```bash
# Windows
start.bat

# Linux/macOS
chmod +x start.sh
./start.sh
```

### Веб-сервер (опционально)

#### Python HTTP Server
```bash
# Python 3
python -m http.server 8000

# Откройте http://localhost:8000
```

#### Node.js HTTP Server
```bash
# Установите http-server
npm install -g http-server

# Запустите
http-server -p 8000

# Откройте http://localhost:8000
```

#### PHP Built-in Server
```bash
php -S localhost:8000

# Откройте http://localhost:8000
```

### Хостинг в интернете

#### GitHub Pages
```bash

   https://username.github.io/repository-name/
```

## 📦 Упаковка проекта

### Создание архива

#### Windows
```bash
# ZIP архив
Compress-Archive -Path * -DestinationPath claude-interface.zip
```

#### Linux/macOS
```bash
# TAR.GZ архив
tar -czf claude-interface.tar.gz *

# ZIP архив
zip -r claude-interface.zip *
```
